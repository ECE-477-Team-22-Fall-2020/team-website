Week 5:

Date: 9/25/2020
Total hours: 6
Description of design efforts:
Assembled and programmed a system for determining the polarity and presence of a magnet using a Hall Effect Sensor placed above it. The sensor's analog output is read by the ADC of the STM32, and the 12-bit output is compared to previously determined integers to see whether it is noise, positive magnet polarity, or negative magnet polarity. Based on some rudimentary testing with the single Hall Effect sensor and a magnet, when the ADC reads below 200 it is negative polarity, and when it reads above 2000 it is a positive polarity. Those ranges will be reevaluated once more Hall Effect sensors to account for individual differences in sensors. After verifying that one Hall Effect sensor could accurately determine the team of a piece, a schematic was created for a 3x3 network of Hall Effect sensors, to be implemeted and tested soon. Below can be found both the code used to initialize and run the ADC during the single sensor tests, as well as the schematic for the 3x3 network.
